# 🚇 Metro Ticket Booking System
--------------------------------

## Project Description
A simple Metro Ticket Booking System built using C# and Entity Framework Core.

## Technologies Used
- C#
- .NET Console App
- Entity Framework Core
- SQL Server
- LINQ

##  Features
- Train and Station management
- Ticket booking system
- One-to-Many relationships
- LINQ queries
- Include() for related data
- Console table output

##  Learning Outcomes
- EF Core Code First
- Database migrations
- Relationships handling
- LINQ queries
- Clean console UI
